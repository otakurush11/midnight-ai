# Midnight AI

Midnight AI is a local-first multi-agent AI companion system built to run on constrained hardware without cloud dependency.

## Purpose
The goal of Midnight AI is to explore how AI systems can act as thinking partners rather than just chatbots.

## Features
- Multi-agent architecture
- Local-first execution
- Memory system
- Tool/plugin system
- Task planning
- Reflection and self-improvement loop
- FastAPI backend
- Ollama models
- React / Three.js interface

## Philosophy
I learn by building systems, breaking them, and rebuilding them better.  
Midnight AI is an experiment in AI architecture, human-AI collaboration, and systems that help people learn and build.
